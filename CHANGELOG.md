# Changelog

## [0.0.1] - 2026-06-14

### 主题功能

- B站粉蓝配色方案（动漫国风）
- 左侧悬浮目录导航（点击弹出，支持 h2/h3）
- 响应式设计，支持移动端
- 表格数据展示优化（渐变表头、斑马纹、hover 高亮）
- 图表展示支持（图片自适应）
- 宽屏布局（最大宽度 1320px）
- 顶部横幅渐变背景，标题居中显示
- 页脚显示仓库信息和主题归属

### 配置

- `guofeng.show_toc` - 是否显示目录导航
- `guofeng.wide_layout` - 是否使用宽屏布局
- 支持 `jekyll-seo-tag` 插件
